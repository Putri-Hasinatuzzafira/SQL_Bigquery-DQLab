# SQL Data Analysis with BigQuery

## Overview
This project showcases SQL-based data analysis using Google BigQuery. It focuses on how SQL queries are used to extract, process, and analyze data efficiently.

## Tools & Technologies
* Google BigQuery
* Google Colab
* SQL

## Topics Covered
* SELECT, LIMIT, ORDER BY
* Filtering (WHERE)
* Aggregation (GROUP BY, HAVING)
* Subqueries & Date Formatting
* JOIN (multiple tables)

## Data Source
The dataset is stored in Google BigQuery (imported from parquet files).

## Project Structure
* 01_select_limit_orderby.ipynb → Basic queries
* 02_conditions_aggregation.ipynb → Filtering & aggregation
* 03_subquery_dateformat.ipynb → Subqueries & date handling
* 04_join.ipynb → Relational analysis using JOIN

## Key Skills
* Data querying
* Data aggregation
* Relational database analysis
* Analytical thinking

## Notes
All queries are executed using Google BigQuery through Google Colab.
